# Cape Town Rental Bot

Проверяет Property24 раз в час и шлёт в Telegram новые объявления
в безопасных районах рядом с Mavericks (Green Point, Sea Point,
Vredehoek, City Centre, De Waterkant), которые подходят под критерии:

- ≤ R20 000 / мес
- 1–2 спальни, 1–2 санузла, 1–2 паркоместа
- аренда на 12 месяцев
- меблированная
- дома и таунхаусы отмечаются ⭐ и идут первыми

## Как запустить (бесплатно, через GitHub Actions)

1. **Создайте аккаунт на github.com**, если ещё нет (бесплатно).

2. **Создайте новый репозиторий** (New repository), сделайте его **приватным**
   (Private) — чтобы никто посторонний не видел код и настройки.

3. **Загрузите туда файлы** из этой папки, сохранив структуру:
   ```
   bot.py
   requirements.txt
   .github/workflows/rental-bot.yml
   ```
   Проще всего — через веб-интерфейс GitHub: "Add file" → "Upload files",
   перетащите все файлы (папку .github тоже, GitHub сам разложит по путям).

4. **Добавьте секреты** (это где хранится токен бота, чтобы он не был виден
   в коде): в репозитории откройте
   `Settings → Secrets and variables → Actions → New repository secret`
   и добавьте два секрета:
   - `TELEGRAM_TOKEN` — токен вашего бота от BotFather
   - `TELEGRAM_CHAT_ID` — ваш chat_id

5. **Готово.** GitHub будет запускать бота автоматически каждый час.
   Проверить вручную можно во вкладке **Actions** репозитория — кнопка
   "Run workflow" запускает проверку прямо сейчас, не дожидаясь часа.

## Настройки

Все параметры — в начале файла `bot.py`, в блоке `CONFIG`:
цена, число комнат/санузлов/паркомест, срок аренды, требование мебели,
список районов и приоритетные типы недвижимости. Меняете значения —
коммитите изменения — при следующем запуске бот уже использует новые критерии.

## Важно

- Property24 может со временем немного менять вёрстку страниц — если бот
  вдруг перестанет находить объявления, скорее всего потребуется поправить
  регулярные выражения в `bot.py` под новую структуру страницы.
- Бот запоминает уже отправленные объявления в файле `seen_listings.json`
  (создаётся и обновляется автоматически), чтобы не слать повторно.
